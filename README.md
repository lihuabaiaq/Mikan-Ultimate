# 💗 刷题鞭策酱

## 📝 项目介绍
这是一个超可爱的 Chrome 扩展程序，专门为 LeetCode 刷题的小伙伴设计～ 
本程序改自[Mikan 刷题鞭策酱](https://github.com/chika-rikka/Mikan) ，多多支持他(她)哦！

当你在 LeetCode 中国站刷题时，这个贴心的小助手会在你提交错误的时候给予温柔的鞭策，让刷题变得更有趣呢！

## ✨ 特性
- 🎯 自动检测 LeetCode 运行与提交状态
- 💝 提供可爱的鼓励提示
- 🎨 精美的界面设计
- 🔔 及时的反馈机制
- ⚙️ 自定义参数设置支持
- ⚠️ 多样且刺激的惩罚机制

## 🚀 快速开始

### 安装方法
1. 在本项目的 Github 页面下载压缩包
2. 解压下载的压缩包到本地文件夹
3. 打开 Chrome 或 Edge浏览器，进入扩展程序页面
4. 开启右上角的"开发者模式"
5. 点击"加载已解压的扩展程序"
6. 选择刚才解压出来的文件夹

### 使用方法
1. 安装完成后，你可以在 Chrome 工具栏看到一个可爱的图标
2. 首先要部署 WebSocket 服务端，就是这个：[websocketNode.js](https://github.com/DG-LAB-OPENSOURCE/DG-LAB-OPENSOURCE/blob/main/socket/BackEnd(Node)/websocketNode.js)(不仅要npm i ws,还要npm i uuid哦～，官方文档杂鱼喵)
3. 点击图标，在弹出的设置页面中添加 WebSocket 连接地址：
   - 如果部署在公网的话：填写 `ws://公网IP:端口` 
   - 如果在局域网中：填写 `ws://局域网地址:端口` (不能用 localhost 或 127.0.0.1 呢～)
   - 如果在自己电脑上，就是设备所连网络的IP地址呢，端口填9999就好啦！
   - 点击保存后，使用设备扫描二维码就能连接啦！

4. 访问 [LeetCode 中国站](https://leetcode.cn)，开始愉快的刷题之旅吧～
5. 开始你的刷题之旅吧！当你提交代码时，鞭策酱会陪在你身边哦～

## 🎉 更新内容
- 本程序改自[Mikan 刷题鞭策酱](https://github.com/chika-rikka/Mikan)  
我们对这个“刷题鞭策酱”做了超级～多厉害的升级哦！ 它现在变得更聪明、更贴心（也更严格！），能满足更多宝宝的💗 刷题鞭策酱！

⚙️ 参数与玩法调整：您现在是主人啦！

1. 全功能设置中心！  
我们变出了一个全新的设置页面 (settings.html)！现在可以通过点击弹窗 下面的“⚙️ 高级设置”按钮 进入。  
从自动增长的间隔、惩罚的强度、持续时间，到奖励扣多少、要不要开启“暂停”按钮...所有的数字和开关，现在都由你说了算！

😈 惩罚与奖励：更“懂”你了哦～

2. 惩罚“叠加”啦！  
以前做错了，惩罚是变成一个固定值。现在是*在你当前的基础上“加上”惩罚值，结束后还要再永久“加上”*一点点（这个“一点点”也可以在设置里改哦）！

3. 独立的“提交失败”惩罚！  
运行与提交失败不同惩罚！如果你点提交却错了，会立刻给你一个严厉的惩罚捏～。  
但它很温柔，不会影响正常的“运行失败”等级计数呢！ 

4. 奖励也“升级”了！  
“提交成功”现在会触发“刷新”效果，让强度立刻归零！(猜猜下一秒强度会直接归零还是飙到200呢，杂鱼喵～)  
“运行成功”会减少强度，你可以在设置里决定要减多少，以及是不是“每题只生效一次”～

5. 每道题强度“快启动”
鞭策酱现在会“快速启动”，“前六次”强度自然增长会更快哦♪  
还可设置初始强度，让你时时刻刻都能感受到鞭策酱的存在呢～

6. 左右通道强度“独立”选择  
现在可以独立控制左右通道的强度提高比率，让左右通道的强度“更”“不同”！  
更有“随机”选项设置，下一次会增加多少呢，猜猜看吧♪

✨ 系统优化：更智能，更贴心！

7. 更好的“换题”识别！  
鞭策酱现在更懂“换题”啦！她会偷偷观察一个神秘请求。一旦发现（代表你切换了题目），就会立刻重置所有状态，不用再一直刷新页面啦！  
这包括“前六次”计数器，让每道新题都能享受“预热”服务！  

8. 浮窗“停止”按钮！  
我们在 LeetCode 页面上的浮窗 加了一个“停止/恢复”按钮（如果你在设置中开启了它）。  
点一下，计时器和加S的周期都会乖乖暂停，强度立刻归零，让你可以中场休息一下～

9. 浮窗“未连接”状态！  
浮窗现在变得更聪明了！在WS和设备都握手成功之前，它会乖乖地藏起所有数字，只显示“🔌未连接...”。直到一切就绪，它才会“啪”地一下显示出强度和计时器！  
浮窗现在可以拖动或者最小化，不用担心挡字了喵！

总之～它从一个固定的小玩具，变成了一个完全属于你、可以随心所欲调教的“小恶魔”哦♪
